API specification for Coinflow

Pages

GET /

Landing/Main Page
Takes in no arguments
Returns html file
Returns errors as json

User Stuff

These are currently usable by everyone, but will at some point require authentication

GET /api/user

Fetch UUID of all users
Takes in no arguments
Returns UUID as json
Returns errors as json

POST /api/user/new

Add new user to database
Takes username, email and password_hash as json in the body
Returns UUID of new user as json
Returns errors as json

GET /api/user/:uuid

Get user info from database
Takes in a UUID in the URI
Returns username, email and password_hash as json
Returns errors as json

PUT /api/user/:uuid

Add new user to database
Takes in a UUID in the URI
Takes new username, email and password_hash as json in the body
Returns updated username, email and password_hash as json
Returns errors as json

DELETE /api/user/:uuid

Remove user from database
Takes in a UUID in the URI
Returns deleted username, email and password_hash as json
Returns errors as json

POST /api/user/login

Create seesion for user
Takes in an email and password_hash in the body
Returns token and uuid as json
Returns errors as json

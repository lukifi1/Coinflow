API specification for Coinflow

User Stuff

GET /api/user

This is currently usable by everyone, but will require need authentication
Fetch UUID of all users
Takes in no arguments
Returns uuid as json
Returns errors as json

GET /api/user/:uuid

Fetch user info from db as json
Takes in a UUID in the URI
Returns username, email and password_hash as json
Returns errors as json

POST /api/user/new

Add new user to database
Takes username, email and password_hash as json in the body
Returns a UUID as json
Returns errors as json

Misc

GET /api

API specification (the document you are looking at right now)
Takes in no arguments
Returns HTML on success
Returns errors as json

GET /api/healthcheck

Healthcheck for docker and for whoever wants it
